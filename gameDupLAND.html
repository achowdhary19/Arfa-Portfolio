Tiny Games: 8 Ball

Description

These games are tiny because they are sketchy and they are made within the tight technical constraints of their custom hardware. These games are made using puzzlescript, pico-8, and bitsy.

My first PuzzleScript game was inspired by 8-ball, a game I was really into at the time. PuzzleScript's biggest constraint is their custom coding language. The main mechanic you can code is: if a player or object is next to something, something can happen. For example, one of the most famous tile based puzzle games Sokoban has one rule: If a player walks into a crate, then push that crate. This rule aligns with the mechanics of 8-ball and allowed me to create an interesting puzzlified version of 8-ball.
