Overview

I was part of a development team that built web and mobile applications to host Providence medical services' new AR volumetric videos. Providence worked with Intel to create these videos in order to provide a more engaging, interactive way to learn and to enhance their projects. Providence was looking for a way to host the videos on an easy to use application that users could access from their homes. I designed and developed the UI elements for the interface. Using HTML, CSS, and javascript.
