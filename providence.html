Providence

Collaborators: Julien Lynge, Taylor Dupuy, Ryan Casement

❮ ❯

I was part of a development team that built a web and mobile application that Providence could use to host their new AR volumetric videos. Providence worked with Intel to create these videos in order to provide patients a more engaging, interactive, comprehensive way to learn stetches, physical therapy excercises, or learn about ----. After working with Intel to create these videos, Providence was looking for a way to host the videos on an easy to use application that users could learn from in their homes. I helped develop these applications using HTML, CSS, and javascript. I designed and developed the UI elements for the interface.
