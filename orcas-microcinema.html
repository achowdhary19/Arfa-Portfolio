About

Website design and development for an itinerant microcinema on Orcas Island, WA. The site prioritizes clear program communication, accessibility, and low-maintenance updates for a non-technical organizer, while presenting the project as a new space for community-based film programming. In addition to website development, I continue to help with maintenence and programming support for future events.

Key Design Decisions

Program-first structure:

Designed the site around screening schedules and events so visitors can quickly understand what’s playing, when, and where, even as venues change.

Client-managed content system

Built a custom, templated page structure that allows the organizer to update programming, descriptions, and images without touching complex backend settings.

Lightweight, accessible UI

Minimal yet characterful visual system that supports readability, accessibility, and performance, while working within constraints of limited assets and low budget.

Archive structure

Structured past screenings and project history to communicate continuity and cultural value beyond single events.

Results

The resulting site is a clear, navigable home base for Orcas Microcinema events that supports both day-to-day programming and long-term storytelling through post-even documentation. This presents the project as a sustained cultural initiative rather than a one-off event series. The event organizer can independently manage updates with minimal friction.
