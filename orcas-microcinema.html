About

Orcas Microcinema is an itinerant microcinema on Orcas Island, WA, screening films across a rotating set of local venues. I designed and built its website to support clear program communication and easy updates for a non-technical organizer, and continue to provide light maintenance and programming support for future events.

Key Design Decisions

Program-first structure:
I built the site around screening schedules and events, so visitors can quickly see what's playing, when, and where — even as the venue itself changes from screening to screening.
A content system the organizer can run
I built a custom, templated page structure so the organizer can update programming, descriptions, and images without touching backend settings — a real constraint, since there's no dedicated technical support behind this project.
Lightweight, accessible visual system
A minimal but characterful UI, prioritizing readability, accessibility, and performance within a limited asset library and near-zero budget.
Archive structure
I structured past screenings and project history to read as continuity — evidence of a sustained cultural initiative, not a series of one-off events, which matters for how the organization presents itself to funders and new audiences.

Results

A clear, navigable home base for the microcinema's programming, supporting both day-to-day updates and long-term storytelling through documentation of past events. The organizer can manage updates independently with minimal friction.
