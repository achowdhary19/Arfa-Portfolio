About

Dollhouse is an interactive mixed-reality experience that connects a physical wooden dollhouse with its digital counterpart. Players place RFID-tagged dolls on a hidden scanner, each one unlocking a differnet home with with unique stories and environments to explore. Developed using Unity and Arduino with custom serial communication.


My Role: Lead Unity Developer & Game Artist

  • ⁑ Programmed Unity-Arduino communication via RFID technology
  • ⁑ Designed and developed a 2.5D/3D game within a 2D environment
  • ⁑ Created all in-game sprites/assets from photographed physical objects
  • ⁑ Implemented point-and-click + NavMesh navigation system
  • ⁑ Built an interactive dialogue system for storytelling and object interaction
