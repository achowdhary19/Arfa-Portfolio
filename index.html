Dollhouse is an interactive, mixed-reality experience that connects a traditional tabletop dollhouse game with a video game counterpart. Using Unity, Arduino, and RFID technology.

UX design and development for Providence's augmented reality web/mobile application displaying 3D physical therapy videos. Created design system, implemented responsive UI, and built interactive elements using JavaScript and A-frame. Featured by Intel. [ Read more ]

Oblivion is a 3D escape room game currently in early development by Unity Production Foundations. Set in historically rich environments, the game challenges players to solve complex puzzles using tools build in the Middle Ages. [ Read more ]

Site design and build for Orcas Microcinema, a nonprofit itinerant microcinema based in Orcas Island, Washington.

Smelling rain (WIP) is a narrative flatgame made in Unity.

Looking for... (WIP) is an interactive poetry site. Made with html, css, and javascript.

8 Ball is a tiny game made within the tight technical constraints of its custom hardware, Puzzlescript.

Angels (WIP) is a butterfly catching game made in Pico-8.
