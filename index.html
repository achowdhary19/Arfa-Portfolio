Hi, my name is Arfa. I’m a designer and programmer interested in tools that encourage curiosity and deeper engagement with information, environments, and culture. Read more about my skills and interests here.

→ guestbook

The Labyrinth is a narrative web experience for Hyundai Artlab’s commission by artist Kyung-Me. Using motion, pacing, and interaction to express the process of internal transformations.

Designing and developing a spatial data visualization tool for Ecolibrium at the Loisaida Center. Investigating how energy infrastructure, environmental conditions, and urban systems interact at a neighborhood scale.

AR Worldcraft is a game design and development course taught at NuVu innovation school.

Redesigned the website for Cafe Mornings to improve navigation, clarify offerings, and support user decision-making for first-time visitors. Structured content and interaction patterns to better connect users with menu, market, and visit information.

Designed the user interface and interaction system for Providence's AR-based physical therapy tool, enabling patients to view and learn exercises in their own space through volumetric video.

Dollhouse is an interactive, mixed-reality experience that connects a traditional tabletop dollhouse game with a video game counterpart. Using Unity, Arduino, and RFID technology.

Oblivion is a 3D escape room game currently in early development by Unity Production Foundations. Set in historically rich environments, the game challenges players to solve complex puzzles using tools build in the Middle Ages.

Website design and development for Orcas Microcinema, a nonprofit itinerant microcinema based in Orcas Island, Washington.

Smelling rain (WIP) is a narrative flatgame made in Unity.

Looking for... (WIP) is an interactive poetry site. Made with html, css, and javascript.

8 Ball is a tiny game made within the tight technical constraints of its custom hardware, Puzzlescript.
